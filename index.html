# L7_React - Todo List Application React-приложение для управления задачами, созданное с использованием Vite. ## Установка 1. Установите зависимости: ```bash npm install ``` ## Запуск Запустите проект в режиме разработки: ```bash npm run dev ``` Приложение откроется по адресу: http://localhost:5173 ## Функционал ### Основные возможности: - ✅ Просмотр всех задач - ✅ Фильтрация по статусам (Все/Активные/Выполненные) - ✅ Добавление новых задач через модальное окно - ✅ Удаление задач - ✅ Редактирование задач (inCell editing): - Клик по описанию - редактирование текста - Клик по статусу - выбор из выпадающего списка - Клик по дедлайну - выбор даты - ✅ Валидация полей (все поля обязательны) - ✅ Подсветка просроченных дедлайнов красным цветом ### Статусы задач: - **Задача выполнена** (зеленый фон) - **Активная задача** (голубой фон) - **Задача отменена** (розовый фон) ### Фильтры: - **Все задачи** - показывает все задачи - **Активные задачи** - показывает только активные задачи - **Выполненные задачи** - показывает выполненные и отмененные задачи ## Технологии - React 18 - Vite 5 - CSS3 ## Структура проекта ``` todo-app/ ├── src/ │ ├── App.jsx # Основной компонент │ ├── App.css # Стили │ └── main.jsx # Точка входа ├── index.html ├── vite.config.js └── package.json ``` ## Особенности реализации - Использован React Hooks (useState) для управления состоянием - InCell редактирование с сохранением по потере фокуса или Enter - Защита от сохранения пустых значений - Автоматическая проверка дедлайнов с подсветкой просроченных - Модальное окно для добавления задач с валидацией - Простой и понятный код без излишней сложности
